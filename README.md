### Installation

1. Clone the repo
2. Open the project folder
3. Now open MongoDB
4. Create a database collection
5. Copy connection url
6. Now run
   ```sh
   cd server
   ```
7. Create .env file
8. Paste the connection url of DB with key name = MONGO_URL
9. Now run commands
   ```sh
   npm i
   nodemon server.js
   ```
10. Now open new terminal
11. Now run commands
    ```sh
    cd client
    npm i
    npm start
    ```
